云服务

ProgramSpace团队为本团队的项目以及所有开发者提供静态云存储服务，这将大大提高空间利用率，避免反复下载/部署字体类的无用文件。

速度

云存储服务的响应速度和下载速度极快，保证了即使在网络不畅通的情况下，也让您的网页/应用程序快如闪电！
在全国各地区响应速度均不超过100毫秒（ms），港澳台地区响应速度甚至一度低于50毫秒，海外速率测试也表现优异。测试字体文件全国任意地区下载时长不超过3秒（s），大部分地区小于1秒，山东、天津、广东等地下载时长低于0.5秒。

部署

要将我们提供的字体等文件部署在您的网页/应用程序？
以测试字体文件（TypeFont）为例。要作为网站字体使用，在CSS的代码中添加： 

@font-face {
    font-family: def-font;
    src: url(https://ProgramSpaceTeam.github.io/cloud/font/TypeFont.woff2) format('woff2');
}

body {
    font-family: def-font;
}
