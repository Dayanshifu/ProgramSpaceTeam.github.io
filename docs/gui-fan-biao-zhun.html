规范标准

这是规范标准文档，所有的（ProgramSpaceTeam）开发者需要遵守规范标准，为提高代码的可读性、利用率、可维护性等。

团队名称

1. 使用 ProgramSpaceTeam（英文）或 ProgramSpace团队（中文）署名，用于版权声明、落款等。
2. 使用 ProgramSpace 用于项目介绍或项目（直接）名称。
3. 使用 ProgramSpaceStudio（英文）或 程序太空工作室（中文）用于团队标准名称，但一般不会在项目或常用显眼处用到。

资源利用

重复的资源将放置于ProgramSpace Cloud里，重复调用，避免空间浪费。

样式标准

1. 在表示名称或特殊含义的英文（如ProgramSpaceTeam）与中文间空一格。
2. 在特殊标点符号（/、#、@、. 等）与其他字符空一格。
